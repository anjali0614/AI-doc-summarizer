# AI Document Summarization Tool

This project implements an LLM-based document summarization pipeline that generates concise, structured summaries from long-form documents.

## Features
- PDF and text document ingestion
- Lightweight text preprocessing
- Token-aware chunking to handle large documents
- LLM-powered summarization with structured outputs

## Tech Stack
- Python
- LLM APIs
- NLP
- PyPDF2, tiktoken
